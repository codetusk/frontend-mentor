### What I learned

It was incredibly difficult for me to nail the semantic HTML on this exercise
until I arrived at this solution, which to me seems pretty defensible.
I also focused on how some of the ARIA attributes work, even where they weren't
strictly necessary for this exercise, and got acquainted with the landmark
system and how landmarks are promoted.

On the CSS side I really needed a brush up, because I had forgotten how almost
everything worked — things like the box model and the basics of building
responsive components.

### Continued development

From here on out, the most important thing for me to study is definitely how to
position objects on a page — more specifically, CSS Grid and Flexbox.

### AI Collaboration

I've used Claude in order to have a back and forth about semantic html and
throw ideas around while debating why something should have X tag instead of
Y.

I've been previously warned about using AI to create semantic and accessible
HTML, and that is why I took this approach of talking about the documented use
cases and making it be some sort of devil's advocate.
